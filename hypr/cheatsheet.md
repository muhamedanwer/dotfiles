# Hyprland Focus & Performance Cheat Sheet

## Workspace Keybinds
| Keybind | Action |
|---------|--------|
| `Super + 1-9` | Switch to workspace |
| `Super + Alt + 1-5` | Instant workspace switch (no animation) |
| `Super + Shift + 1-9` | Move window to workspace |
| `Super + ,` | Previous workspace |

## Window Focus
| Keybind | Action |
|---------|--------|
| `Super + Tab` | Cycle next window |
| `Super + Shift + Tab` | Cycle previous window |
| `Super + P` | Pin window (stays on all workspaces) |
| `Super + H/J/K/L` | Focus left/down/up/right |

## Window Management
| Keybind | Action |
|---------|--------|
| `Super + Q` | Close window |
| `Super + T` | Toggle floating |
| `Super + F` | Fullscreen |
| `Super + G` | Toggle group |
| `Super + F11` | Toggle split |

## Apps
| Keybind | Action |
|---------|--------|
| `Super + Return` | Terminal (alacritty) |
| `Super + Space` | App launcher (wofi) |
| `Super + E` | File manager (dolphin) |
| `Super + W` | Browser (chromium) |
| `Super + C` | VS Code |
| `Super + R` | Terminal + ranger |

## System
| Keybind | Action |
|---------|--------|
| `Super + X` | Lock screen |
| `Super + S` | Screenshot (select) |
| `Super + Shift + S` | Screenshot (clipboard) |
| `Super + V` | Clipboard menu |

## Performance Optimizations Applied
- Animations disabled (instant response)
- Blur disabled (GPU savings)
- Flat mouse acceleration (consistent tracking)
- Fast key repeat (80 rate, 200ms delay)
- Focus follows mouse enabled
